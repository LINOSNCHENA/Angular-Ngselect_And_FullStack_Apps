## Dev

```
  npm install @vue/cli-service -g
  npm i bootstrap jquery popper.js --save
  npm i bootstrap-vue --save
  npm install --save vue vue-router
  npm i vue-template-compiler --save
  npx npm-check-updates -u
  npm install

```
